Introducing scala and sbt

Using scala and sbt is somehow tricky at the beginning. I just sum up my experience with it compacting the different steps to make it run. I will describe the setup for MacOSX.

Install the prerequisites:

you need the following tools

  • Emacs
  • sbt
  • scala

type in a terminal 

bash >> brew install emacs
bash >> brew install sbt
bash >> brew install scala

Configure emacs

then, you must get the following Emacs packages and install them from Emacs

  • scala-mode2
  • ensime

configure them by creating or modifiying the init.el file where you add 

;; setup Scala
(require 'scala-mode2)

(require 'ensime)
(add-hook 'scala-mode-hook 'ensime-scala-mode-hook)

;; setup Magit (could be required)
setq magit-last-seen-setup-instructions "1.4.0")

Configure sbt to use ensime

add a line into the plugin.sbt located in the folder .sbt/plugins. If not there, you may create one. 

bash >> mkdir -p .sbt/plugins/plugin.sbt

and add the following line with the version of the plugin (at least 0.1.0) 

addSbtPlugin("org.ensime" % "ensime-sbt-cmd" % "0.1.0")

Create the scala project

now you should be ready to create a Scala project, prepare the project directory tree, basically you must follow a Maven directory structure 

bash >> mkdir -p projectsample/project
bash >> mkdir -p projectsample/src/{test,main}/scala

write a simple Main.scala in the src/main/scala folder 

package com.projectsample

object Main  {
 def main (args : Array[String]) = {
   println("Scalala")
   }

 }

Use sbt and ensime

run sbt under the root of the project 

bash >> sbt

then in the sbt shell, type the command gen-ensime that will generate the configuration 

sbt >> gen-ensime

once done open the Main.scala in emacs, activate ensime from Emacs do Meta-x (Esc-x) and type 

: ensime

And that's it!
